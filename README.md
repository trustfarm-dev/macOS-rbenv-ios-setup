# 🍎 rbenv-ios-setup

최신 iOS 및 React Native 개발환경을 macOS에 한 줄로 세팅하는 자동 설치 스크립트입니다.

## 📦 설치 방법

```bash
git clone https://github.com/yourname/rbenv-ios-setup.git
cd rbenv-ios-setup
chmod +x install.sh
./install.sh
```

## 💎 포함 구성

- Xcode CLI
- Homebrew 기반 openssl/libyaml/nvm
- rbenv 최신 GitHub clone
- ruby 3.2.2
- nvm + Node 18
- CocoaPods, bundler
- React Native CLI
